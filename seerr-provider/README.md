# Seerr Noctalia Plugin
Search and request movies and TV shows from your Overseerr or Jellyseerr instance directly from the Noctalia launcher.

## Configuration
Go to plugin settings and enter your Seerr `Server URL` and `API Key`.

## Usage
Open your launcher and type `>seerr ` followed by your search query.
Select the movie or TV show, then pick which server and profile you want to request it on.
