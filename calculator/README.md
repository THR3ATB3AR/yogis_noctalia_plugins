# Calculator Plugin

A simple calculator plugin for the Noctalia launcher.

## Features

- **Inline Calculations:** Just type your math expression (e.g. `2 + 2`) into the launcher, and the result will instantly appear.
- **Forced Evaluation:** Prefix any expression with `=` to force evaluation (e.g. `= Math.sqrt(16)` or `= 5 * 12`).
- **Clipboard Integration:** Hit enter on the result to instantly copy the calculated value to your clipboard.

## Requirements
- `wl-copy` (for copying results to the clipboard under Wayland).

## Installation

Load the plugin directory into your Noctalia instance, enable it, and use the launcher.
